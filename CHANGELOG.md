# Historial de versiones

## [1.0.0] - 2025-05-26
### Añadido
- Juego base con sistema de oleadas.
- 8 tipos de zombies (normal, lento, esquivador, explosivo, fuego, hielo, veneno, lanzador).
- 14 armas (francotirador, ballesta, arco, rifle, escopeta, subfusil, lanzallamas, doble pistola, cuchillo, guadaña, hacha, martillo, katana).
- Sistema de mejoras al subir nivel (5 mejoras genéricas con rareza).
- Power-ups (velocidad, escudo, daño, regeneración, vampiro, imán, invencibilidad).
- Logros desbloqueables (6 logros).
- Interfaz de usuario con ajustes de escala, sangre y números de daño.
- Menú de pausa con estadísticas del jugador y armas.
- Hitbox cuadrada para el jugador.
- Rotación del personaje hacia el ratón.
