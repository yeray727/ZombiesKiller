# Manual de Usuario - ZombiesKiller

## Introducción

**ZombiesKiller** es un juego de acción por oleadas donde controlas a un héroe que debe sobrevivir a hordas de zombies. Cada oleada es más difícil, y al acumular XP puedes subir de nivel y elegir mejoras permanentes.

## Inicio del juego

1. Al abrir el juego, se muestra el menú principal.
2. Selecciona tus armas iniciales (una de cada categoría: largo alcance, corto alcance y cuerpo a cuerpo).
3. Haz clic en **COMENZAR**.

## Controles

| Acción               | Tecla / Ratón               |
|----------------------|-----------------------------|
| Moverse              | `WASD` o `Flechas`          |
| Disparar / Atacar    | Mantener **clic izquierdo** |
| Cambiar arma         | `1` (largo), `2` (corto), `3` (cuerpo a cuerpo) |
| Dash (esquive rápido)| `Shift`                     |
| Patada               | `C`                         |
| Pausa / Menú         | `ESC`                       |

## Barra de estado (HUD)

- **Oleada**: Número de ronda actual.
- **🧟**: Número de zombies vivos.
- **🪙**: Monedas acumuladas.
- **❤️**: Vida del jugador.
- **⭐ Nivel**: Nivel actual y barra de XP.
- **Arma activa**: Nombre del arma que estás usando.
- **💥 Puntuación**: Puntos totales.

## Subir de nivel y mejoras

Cada vez que matas zombies, ganas XP. Al llenar la barra de XP, subes de nivel automáticamente al final de la oleada y se abre un menú con 3 mejoras aleatorias (con rareza: común, poco común, raro, épico, legendario). Elige una para mejorar permanentemente tus estadísticas.

## Tipos de zombies

| Zombie       | Característica                                                                 |
|--------------|--------------------------------------------------------------------------------|
| Normal       | Movimiento y vida estándar.                                                    |
| Lento        | Más vida, más lento.                                                           |
| Esquivador   | Rápido y se mueve en zigzag.                                                   |
| Explosivo    | Al morir explota dañando área.                                                 |
| Fuego        | Aplica quemadura al contacto.                                                  |
| Hielo        | Ralentiza al jugador.                                                          |
| Veneno       | Envenena al jugador.                                                           |
| Lanzador     | Mantiene la distancia y dispara proyectiles.                                   |

## Power-ups

Al matar zombies, pueden caer power-ups con estos efectos:

- 🟢 **Velocidad**: Aumenta temporalmente la velocidad.
- 🔵 **Escudo**: Absorbe daño.
- 🔴 **Daño**: Aumenta el daño.
- 💗 **Regeneración**: Restaura vida con el tiempo.
- 🩸 **Vampiro**: Roba vida al matar.
- 🧲 **Imán**: Atrae XP y power-ups.
- ✨ **Invencibilidad**: Invulnerable durante unos segundos.

## Menú de pausa

Pulsa `ESC` para abrir el menú de pausa. Desde allí puedes:

- Ver tus estadísticas actuales (vida, defensa, velocidad, daño, cadencia, crítico, regeneración, escudo, XP).
- Ver los logros desbloqueados y el progreso de los restantes.
- Activar/desactivar la sangre y los números de daño.
- Ajustar el tamaño de la interfaz (UI scale).
- Reanudar el juego.

## Fin del juego

Cuando tu vida llega a 0, aparece la pantalla de "Game Over" con tu oleada final, puntuación y nivel. Puedes reiniciar o volver al menú principal.

## Consejos

- **Mantente en movimiento** para evitar ser rodeado.
- **Cambia de arma** según la situación: usa escopeta o lanzallamas para grupos, francotirador para zombies lejanos.
- **Prioriza mejoras defensivas** en oleadas altas (defensa, escudo, regeneración).
- **Los power-ups de vampiro e imán** son muy útiles para recuperar vida y XP rápidamente.
